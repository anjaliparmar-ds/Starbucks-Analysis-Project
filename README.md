Starbucks Analysis Project

☕ Starbucks Beverage Analysis Dashboard | Power BI

📊 Project Overview
This project is an interactive Power BI dashboard built to analyze Starbucks beverage data. The dashboard provides insights into beverage categories, caffeine content, calories, sugar levels, and protein values using interactive visualizations and filters.
The goal of this project is to demonstrate data visualization, dashboard design, and business intelligence skills using Power BI.

🚀 Dashboard Features

📌 KPI Cards
Total Beverages
Average Calories
Average Sugar
Average Caffeine


📈 Interactive Visualizations
Average Calories by Beverage Category
Average Caffeine by Category
Beverage Category Distribution
Top 5 Highest Caffeine Beverages


🎛 Interactive Filters
Protein Range Slider
Beverage Preparation Filter

🛠 Tools & Technologies
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
- Data Visualization

📌 Key Insights
- Coffee beverages have the highest average caffeine content.
- Smoothies contain the highest average calories.
- Beverage categories show varying sugar and calorie levels.
- Interactive filters allow users to explore beverages based on protein range and preparation type.
